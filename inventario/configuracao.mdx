## Ferramentas

No setup atual está sendo utilizado as seguintes ferramentas para desenvolvimento do app:

- Visual studio code: codificação da aplicação
- Android studio: emulador android e geração do `.apk`
- Postman: documentação da API `VTCONN > Inventário`

## Tecnologias

Para criar o app estamos utilizando:

- React native
- Expo
- NativeWind CSS
- Axios
